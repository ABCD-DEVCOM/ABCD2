306Duración de Material (NR O)

Seis caracteres numéricos, con el patrón hhmmss, que representan el tiempo de duración para una grabación sonora o la duración de una actuación para música impresa o manuscrita. Si el tiempo de duración es menor a una jora, la hora (hh) se llena con ceros; si es menos de un minuto, los minutos (mm) se llenan con ceros.

Indicadores:

- En Blanco (_)

Subcampos:

^a Tiempo de duración (R)

Ejemplos:

306 __^a002016

306 __^a003100 ^a001839
