Etiqueta

45

Nombre

Símbolo (Naciones Unidas)

Definición del dato

Signatura que usan algunos organismos para identificar sus publicaciones.

Propósito del campo

Este campo se utiliza para consignar el símbolo de un documento tratado a nivel analítico, monográfico o colección.

Descripción de los elementos

Ocurrencias múltiples. El símbolo es una combinación de números y letras, y debe registrarse tal como aparece en la publicación.
Los símbolos no deben confundirse con los números de venta, de informe, de proyecto, etc. Las siglas o un conjunto de siglas no constituyen símbolos.
Cuando el documento presenta más de un símbolo, ellos deben registrarse separados entre sí por el elemento que indica las ocurrencias múltiples.

Ejemplo

LC/L.1697/Rev.1
LC/G.2194-P
LC/R.2101
