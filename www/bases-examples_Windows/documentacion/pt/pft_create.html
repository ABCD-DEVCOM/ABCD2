Criar formato de exibição

De acordo com a Tabela de definição de campos, é exibida a lista de campos existentes.

Selecione na janela da esquerda os campos que serão utilizados no formato de exibição e use as setas de direção para passá-los à janela da direita. Use os links Subir e Descer para mudar a ordem dos campos no formato que será criado.

Terminada a seleção, indique se deseja criar a saída na forma de tabela (separando os datos em linhas e colunas) ou na forma de parágrafo (separando os dados com um salto de linha). Será exibido na janela o formato criado.

Para salvar o formato criado clique no link Salvar formato
