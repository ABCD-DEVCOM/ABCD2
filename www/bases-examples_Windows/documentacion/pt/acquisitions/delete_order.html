Esta opção apaga uma ordem de pagamento já emitida.

Para que esta ação possa ser executada é necesario apagar a sugestão asociada aos dados correspondentes a ordem  emitida.

A ordem de execução do processo é a seguinte:

  1. Acessar as sugestão que estão relacionadas com a ordem de compra
  2. Eliminar nas sugestões os dados da ordem de compra
  3. Repetir esta ação para todas as sugestões associadas com a ordem de compra 

Uma vez que não exista vinculação entre a ordem de compra e as sugestão, então se permite a eliminação da ordem de compra.
