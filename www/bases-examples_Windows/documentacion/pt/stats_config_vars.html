Configuração de estatísticas - Lista de variáveis

Esta opção permite definir as variáveis de uma base de dados que podem ser utilizadas para a emissão de estatísticas.

São necessárias três colunas de informações:

Campo Define o título do campo, tal qual como se reportará nas linhas ou colunas da tabela de frequência ou de contingência
Formato de
Extração		 O pft a aplicar para extrair o conteúdo do campo da base de dados, a fim de refletir na tabela de frequência ou de contingência. O valor produzido pelo PFT será utilizado como cabeçalho da linha ou coluna correspondente e será acumulado para efeitos de totalização
 Prefixo  (Por implementar) Permite extrair as variáveis diretamente do arquivo invertido, reduzindo portanto o tempo de leitura dos dados
   

Este arquivo é chamado de stats.cfg e é armazenado na pasta def da base de dados correspondente.

As variáveis aqui definidas podem ser utilizadas em dois tipos de processos:

  • Para emitir tabelas de contingência e tabelas de frequência, selecionando as variáveis que estarão nas linhas e as variáveis que estarão nas colunas
  • Para a preparação de tabelas predefinidas onde são emitidos resultados selecionando o nome do quadro correspondente
