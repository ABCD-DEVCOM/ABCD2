Esta janela mostra os registros encontrados no servidor Z39.50 selecionado.

Para passar um registo específico para a base de dados local, clique na imagem mostrada no lado esquerdo de cada registro.  Apagará o registro da área de texto, mas a página com os resultados permanece aberta para continuar permitindo a captura de registros. O registro é capturado na área de entrada da base de dados local, permitindo a edição antes de ser armazenado na forma final.

Se você selecionou um formato de conversão, o mesmo é aplicado durante a opção de captura.

Os registros recuperados utilizam a tabela MARC-8 para representar caracteres especiais ou diacríticos.  A equivalência entre o alfabeto ANSI e os diacríticos é feita sob a opção Configurar cliente z39.50  do menu principal do ABCD.
