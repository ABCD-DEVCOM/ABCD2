Bases de dados de itens disponíveis para a concessão de empréstimos

Bases de dados definidas no ABCD contendo itens disponíveis para a concessão de empréstimos podem ser selecionadas por meio deste procedimento.

Selecione da lista apresentada uma base de dados e depois clique em Ir para definir os parâmetros e formatos requeridos para fazer a ligação entre a base de dados e o sistema de empréstimos.

Em linhas gerais, é necessário definir:

  • A chave de acesso aos títulos
  • Como determinar o total de exemplares do título
  • Como extrair a informação de cada exemplar
  • Como apresentar a informação catalográfica do título
  • A informação do título a transferir para a base de dados de empréstimos, para identificação dos materiais emprestados e para obter as estatísticas
  • Como apresentar a informação do título emprestado, depois de estar armazenado na base de dados dos empréstimos

Após selecionada a base de dados surgirá um formulário de entrada onde se solicitam os parâmetros e formatos requeridos pelo processo.
