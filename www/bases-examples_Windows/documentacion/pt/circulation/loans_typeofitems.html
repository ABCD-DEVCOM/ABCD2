Tipos de materiais

Esta tabela descreve as políticas que serão aplicadas para os diferentes tipos de materiais existentes na base de dados. O tipo de material referenciado na tabela deve ser gerado pelo formato de extração que é aplicado à base de dados com os materiais para empréstimo. Este resultado, entre outros dados, deve conter o tipo de material a ser localizado na tabela para que sejam atribuídas as respectivas políticas.

Se o formato de extração não gerar nenhuma saída, o operador poderá decidir que tipo de empréstimo a ser aplicado, selecionando uma das opções da tabela Tipos de Empréstimo.

Quando se grava o registro de empréstimo, é salvo o tipo de material ou de tipo de empréstimo que foi atribuído para efeitos de aplicar corretamente as políticas de devolução/multa/suspensão
