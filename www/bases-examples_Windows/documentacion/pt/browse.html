Permissões no ABCD 

Os operadores ou usuários do ABCD são as pessoas que têm acesso às diferentes funções e bases de dados definidas no sistema. Existem 8 tipos de operadores:

Administrador do Sistema

Tem acesso a todos os módulos, funções e bases de dados definidos no ABCD. Também é o responsável por definir novos usuários e atribuir-lhes as permissões.

Gestores de Bases de Dados

Podem:

  • Criar, modificar e eliminar a estrutura de bases de dados
  • Realizar todas as funções de administração das bases de dados sob seu controle (modificar a estrutura, FST's, parâmetros de configuração de estatísticas, desbloquear bases de dados e registros, bloquear bases de dados para impedir o registro de informação, exportar e importar registos
  • Realizar processos de alterações globais nas bases de dados sob seu controle
  • Atribuir operadores às bases de dados sob seu controle

Operadores de Bases de Dados

Podem:

  • Realizar funções de criação e edição de registros nas bases de dados sob seu controle
  • Produzir relatórios ou gerar outros serviços nas bases de dados sob seu controle

Gestor de Empréstimos

Pode:

  • Criar/Modificar as políticas de empréstimo da instituição
  • Executar todas as funções relacionadas com o sistema de empréstimos
  • Gerir as bases de dados de transações, reservas, suspensões e multas
  • Eliminar reservas, multas e suspensões

Operadores de Empréstimo

Podem:

  • Realizar todas as transações relacionadas com o serviço
  • Obter relatórios e produtos

Criar Novo Operador

Clique sobre Novo usuário para criar um novo operador de ABCD. Aparecerá uma folha de entrada com os campos de informação requeridos. Complete a informação solicitada e clique sobre:

  • Guardar o registo, ou
  • Cancelar

em função da ação requerida. [inicio]

Editar Operador

Clique no ícone da edição associado ao operador cuja informação deseja editar. Aparecerá o formulário de entrada com a informação correspondente. Edite o registo e ao finalizar clique em guardar ou cancelar dependendo da ação requerida. [inicio]

Eliminar Operador

Clique no ícone de eliminação associado ao operador que deseja eliminar. [inicio]
