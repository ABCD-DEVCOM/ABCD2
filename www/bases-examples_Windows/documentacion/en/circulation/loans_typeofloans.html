Loan types

This table is applied when the extraction format for lendable material does not generate any output to be found in the Types of Material.table.

It will be displayed to the operator, who will determine the policy to be applied.

Field How to apply the policy
Loan type Loan type code
Description Type of loan description
Loan period Number of days/hours to calculate return time.
Unit Loan period measuring unit (days/hours)
Renewals number Number of renewals allowed
Fine per day after due date. Amount of the fine incurred per each day/hour after due date in returning materials.
Number of days of suspension per day after due date It states the number of days the user is to be suspended per each day/hour after due date in returning materials.
