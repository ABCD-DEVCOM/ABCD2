Para solicitar un estado de cuenta, iintroduzca el código del usuario y haga  clic sobre Buscar. Puede utilizar el botón Listar para obtener una lista alfabética de los usuarios registrados.

Si no conoce el código del usuario, puede suministrar el número de inventario de algún objeto que esté en poder del mismo, utilizando para ello el recuadro No. de inventario.  Si hace clic sobre Listar se presentarán los números de inventario de los objetos que actualmente están en poder de algún usuario.
