This presents a window divided into two frames with the following details:

Left frame:

Fields requested for the construction of the advanced search form. This displays the following columns:

Prefix assigned to the field, with the object of presenting the dictionary of terms of a field, or group of fields
Field name As will be shown in the advanced search form
Id. in the FST ID with which the field is indexed in the FST. If various fields are applied in the same search, separate the id's with commas. E.g.: 100,600,700
Prefix

It is important to note that there are two ways to find the terms to include in any search box on the form:

  • If the selection is done through the dictionary of terms (index link), then this is transfered into the search term box with the assigned prefix. If you select more than one term, they are linked with the operator "or". The terms are enclosed in quotation marks ("..") to indicate to the search process that the terms are searched as a concept not as individual words, and to apply them just as they appear in the form. In this case the sub-expression would appear in the form "AU_GARCIA MARQUEZ, GABRIEL" or "AU_VARGAS LLOSA, MARIO"
  • If a word is written directly into a search box in the form, then the search expression linking each word is built with the operator (F) and at the end of the term the operator is constructed by adding the following qualification: (Id. in the FST) using the column mentioned in the advanced search form. For this option to work correctly, the fields should be indexed with technique 4, generating the number of occurrences with the operator '%', when the field is repeatable. In this case the sub-expression is generated as follows:  GARCIA(F)MARQUEZ(F)GABRIEL/(100,600,700) or VARGAS(F)LLOSA(F)MARIO/(100,600,700)

In the FST the following lines are required for indexing:

100 0 "AU_"v100^a
600 0 (|AU_|v600^a|%|/)
700 0 (|AU_|v700^a|%|/)
100 4 v100^a
600 4 (v600^a|%|/)
700 4 (v700^a|%|/)

and the advanced search form will indicate:

Name of field: Personal author
Id. of the FST: 100,600,700
Prefix: AU_

The advanced search form is stored in the folder def of the database, under the name camposbusqueda.tab (Search Field.tab)
