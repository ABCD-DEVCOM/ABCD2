Modify the database definitions

Once a database has been created, it is possible to enter this menu to modify any of the components defined by an ABCD database.

Existing components are:

  • Field definition table. Define the structure of the records in the database 
  • Field selection table. Define the strategy to apply when building indexes to access records in the database 
  • Input worksheets. Additional input formats to be used for entering information 
  • Output Formats. Display formats that can be used to display information
  • Record types. Define the input formats to use for each type of record as defined in the database (if any)
  • Record validation formats. This enables a revision of the consistency of a record prior to updating
  • Advanced Search Format. Define the fields that can be used to format the advanced search 
  • List of available databases. List of databases that appear in the different modules of ABCD 
  • dbn.par file . Access path to the database resources
  • Help files for the fields of database. Online help that can be deployed at the time of entering information
  • Settings under the IAH. Configuring the database for public access
  • List of variables to obtain statistics. List of fields that can be used to obtain statistics, indicating the extraction format of each
  • Statistical tables. Combination of fields in rows and columns to form pre-defined tables
