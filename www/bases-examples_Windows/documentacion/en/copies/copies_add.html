Agregar copias a un objeto

Los objetos a los cuales se le inventarían las copias deben poseer un campo denominado "Número de control" el cual debe estar definido en la FDT e indizado en la FST con el prefijo "CN_".  Este campo de la FDT se utiliza como campo de relación entre un objeto y sus copias.

Las copias de los objetos todas las bases de datos confluyen en una base de datos denominada copies y se mantiene un único número de inventario para cada copia.

Para la asignación secuencial del número de control en cada registro, se utiliza el archivo control_number.cn ubicado en la carpeta data de la base de datos correspondiente. Cada base de datos bibliográfica posee su propia secuencia del número de control. Por esta razón, en la base de datos copies se almacena también el nombre de la  base de datos relacionada con la copia para garantizar la correcta recuperación de la información bibliográfica, cuando ello sea necesario.

Cuando se solicita el ingreso de nuevas copias, se presenta un formulario para ingreso de la información correspondiente.

Si el número de inventario de la base de datos copies se ha definido en la FDT como autoincrement entonces, se presenta en el formulario de ingreso una casilla denominada Número de copias a agregar, a fin de que se informe cuántas copias del mismo objeto se van a crear en  la base de datos copies. Las mismas se agregarán en forma automática, asignandole a cada una un número de inventario que se incrementará desde el último número asignado, tomando como punto de partida el valor almacenado en al archivos control_number.cn almacenado en la carpeta bases/copies/data. Use la opción "Restablecer el número de inventario" del menú principal de catalogación, si por alguna razón se necesita restaurar el próximo número de inventario a asignar a las copias.

Si el número de inventario de la base de datos copies (campo 30) no se ha definido como autoincrement, entonces, si va a ingresar más de una copia, coloque, separados con [Enter] , en la casilla correspondiente al número de inventario, la secuencia de números de inventario a asignar a las copias que se crean. Esto es, si se coloca en la casilla los valores

0011
0022
0035

ABCD creará en la base de datos copies 3 registros a los cuales les asignará respectivamente los números de inventario suministrados.

Use el vínculo Ver todas las copias para desplegar todas las copias ya ingresadas para ese objeto.

Rellene los datos correspondientes y cuando termine haga clic sobre Actualizar.

Se creará en la base de datos copies el número de registros solicitados, de acuerdo al valor Número de copias o  los números de inventario suministrados.
