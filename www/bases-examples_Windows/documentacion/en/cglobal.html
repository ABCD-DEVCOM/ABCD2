Global Changes

To perform global changes follow these steps:

  1. Choose the method for selecting the records

  2. Choose the field where you want to apply the change

  3. Choose the type of change:

    • Add

    • Modify

    • Delete

    • Divide

  1. Enter the actual value to be changed, for example, if you want to delete the field 30, when the field 30 has the value “Medicine”.

  2. If the type of change selected is Modify, select the scope of the modification:

    • Over the whole field

    • Over a portion of the field

  3. Choose the new value to add or modify

  4. If the change is Divide, select:

    • the character or characters to be used as delimiters

    • the field to which the selected portion is going to be placed

    • which portion of the field is going to be moved to the selected field

Record selection method

There are two ways to select the records where the global change is going to be performed:

  1. Giving a range of Mfn

  2. Building a search expression. In this case the advanced search form is displayed in order to build the search expression.

Select the search method which will allow you to select the records where the change is going to take place.

Field where the change is going to be made

The fields defined in the FDT appear in a list in the menu. Select:

Type of change

There are three types of changes:

  1. Add field

  2. Modify the content of a field

  3. Delete a field

The selected change will be applied to the selected field of the records which were retrieved by specifying a range of Mfn or by performing a search.

Scope of the change

If you selected the option Modify the content of a field, you will need to specify if you want to replace the whole content of the field, or if the change is going to be applied to a portion of it, for example, to replace the content of a specific subfield.

Apply a change according to the content of a field

This box lets you select a list of terms that will implement the conditions of the change. That is, if the actual content of the field coincides with at least one of the terms specified, then the change will have effect. If you don't specify any value, then the change is applied without conditions.

New value of the field

Insert the value that you want to add or use as a replacement for the specified field in this box.

Dividing a field

This option lets you insert a punctuation mark or any other delimiter inside a field in order to:

  • Transfer it to another field

  • Delete it

You can select which portion of the field (before or after the delimiter) will be deleted or transferred to another field.

Change implementation

The change starts to take place over groups of records. A link to the records will appear in order to allow revision during the process.

Important note: Be sure to make a back up of the database before performing any global changes.
