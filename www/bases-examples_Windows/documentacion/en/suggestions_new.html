Crear nueva recomendación

Antes de insertar una nueva recomendación en la base de datos, debe verificar si se trata de la adquisición de un nuevo objeto o de copias adicionales a un objeto hya existente.  En este caso debe suministrar la base de datos donde se encuentra la información catalográfica del objeto y el Número de identificación del objeto, tal como fue asignada en la base de datos correspondiente.

Si se trata de un nuevo objeto, también puede suministrar el nombre de la base de datos donde se catalogará el objeto.

Los datos obligatorios a suministrar son:

Etiqueta  
2 Estado de la sugerencia
3 Tipo de adquisición
5 Base de datos donde se cataloga el objeto
6 Número de identificación del objeto (Obligatorio si el tipo de adquisición seleccionado es Copias adicionales)
16/17 Se debe indicar el autor personal o el autor corporativo
18 Título
200 Nombre de la persona/departamente que sugirió la compra del objeto
210 Fecha de la sugerencia
211 Fecha ISO de la sugerencia
220 Número de copias solicitadas

 La información ingresada se almacena en la base de datos suggestions y a los nuevos registros se les asigna en forma automática un número secuencial el cual es utilizado como campo de relación en los procesos de seguimiento del estado de la recomendación.

Para la asignación del número secuencial, se mantiene en la carpeta data de la base de datos suggestions un archivo denominado control_number.cn el cual posee grabado el último número asignado.
