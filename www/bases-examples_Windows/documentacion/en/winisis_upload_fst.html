If the last step was completed, ABCD will have registered the FDT and the FST of the database during the creation of the database.

Use the box Mount .pft to locate the output format to be assigned to the database. Only by doing this can one move a format from the module Structure Maintenance or create additional formats.

Remember that the format selected will be coded in a form which outputs a page in HTML. Formats which do not comply with this condition will generate a syntax error in ABCD.
