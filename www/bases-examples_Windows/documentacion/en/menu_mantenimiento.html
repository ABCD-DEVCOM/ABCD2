Maintenance Functions

These allow you to:

  1. initialise a database

  2. delete a database

  3. lock a database

  4. unlock a database

Initialise a database

This option deletes – in a permanent and unrecoverable way - all the records of a database.

Delete a database

This option deletes - in a permanent and unrecoverable way - all the files of the database, and all the formats and files for the definition of a database.

Lock a database

Using this option the administrator can lock access to a database, be it because it is being reorganized, the indexes are being built, or for any other reason that doesn`t allow data entry.

Unlock a database

Using this option, the administrator can allow access to a database that was locked for maintenance reasons.

Assign control number

By means of this option ABCD can automatically assign a control number that is incremental from an initial value.  The FDT of the database needs to be defined with a field type autoincrement
