Capture records from other databases

Select the database from which to capture the records. Use the pick-list that is in the box Capture from the database: This selection will remain active until you change to another database

Once the database has been selected, the alphabetical list of the main entry of the records in the database which must be specified in the Field Definition Table is displayed in the box on the right. Click on an entry to display details.

To move in the list of records, use the alphabetical index or click on More terms in order to continue reading from the last term displayed in the window. If you want to go to a specific title, put the first letters or words in the window Go to  and press [Enter] or click to continue.

You can use the edge of the frame to enlarge or reduce the width of the windows

If you want to copy the displayed record to the active database, click on to activate the entry mode. You can edit before copying and no changes will be seen reflected in the original record.

Do not close this window if you are going to capture more than one record.

The last database from which the capture of records is executed, remains active during the following capture process.

Conversion Table

You can select a conversion table to convert fields between databases with different structures and make these equivalent. The conversion tables are created or edited under the option Configure Z39.50 client of the main ABCD menu
