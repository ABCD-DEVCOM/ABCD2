Reset the last inventory number or control number
If the database has a field defined for the automatic rate of increase for the purposes of storing a control number for each record, this option allows you to alter the sequence by modifying the value of the last number assigned in the file control_number.cn
If the data base has copy control, the modification of the value of the last number assigned in the inventory of copies is allowed
