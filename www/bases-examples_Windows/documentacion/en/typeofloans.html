Loan Types

This table is applied when the extraction format for the loaned item does not generate an item type which is found in the table Types of Item.

The loans operator is presented with the loan types existing to select the policy to apply.

Field Use in the application of the policy
Loan type Code for the loan type
Description Description of the loan type
Loan period Number of days/hours for calculating the due date
Unit Unit of measurement for the loan period (Days/Hours)
No. of reservations Number of reservations permitted
Fine per day overdue Shows the fine to be charged for each day/hour overdue in the return of a loan 
Days of suspension per day overdue Indicates how many days the user should be suspended for each day/hour overdue in the return
