Importation d'enregistrements d'autres bases de données

Sélectionnez la base de données d'où importer l'enregistrement. Utilisez la liste de choix située dans l'option Importation de la base de données :. Cette sélection reste active jusqu'à ce que vous la changiez pour une autre base de données

Une fois sélectionnée la base de donnée, vous verrez dans la boîte de droite les enregistrements de la base correspondante classés par ordre alphabétique de l'entrée principale, tel que spécifié dans la Table de Définition des Champs. Cliquez sur les entrées pour voir les détails.

Pour se déplacer dans la liste des enregistrements, utilisez l' Index Alphabétique ou cliquez sur Plus de termes pour continuer à les lire à partir du dernier présenté dans la fenêtre. Si vous souhaitez accéder à un enregistrement spécifique, inscrivez ses premières lettres ou mots dans la fenêtre Aller à  et appuyer sur [Entrer] ou cliquez sur Continuer.

Vous pouvez utiliser le bord intérieur des cadres pour étendre ou réduire la taille des fenêtres

Si vous voulez copier l'enregistrement en cours vers la base active, cliquez sur pour activer le mode de saisie. Vous pouvez éditer les champs avant de copier l'enregistrement, sans que les changements n'aient d'effet sur l'enregistrement original.

Ne fermez pas cette fenêtre si vous importez plusieurs enregistrements.

La dernière base utilisée pour l'importation d'enregistrements restera active pendant le processus d'importation.
