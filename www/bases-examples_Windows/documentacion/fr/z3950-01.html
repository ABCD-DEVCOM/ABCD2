Les copies locales des enregistrements en provenance du serveur Z39.50 sélectionné sont présentées dans cette fenêtre.

Pour transférer un enregistrement spécifique à la base de données locale, cliquez sur l'image qui est affichée dans la marge gauche de chaque enregistrement.  L'enregistrement est retiré de la zone de texte mais la page avec les résultats reste ouverte pour permettre la capture des enregistrements. L'enregistrement capturé est affiché dans la zone de texte de la base de données pour permettre son édition avant sa sauvegarde dans le formulaire définitif.

Si un format de conversion est sélectionné, le même est appliqué pendant l'option de capture.


Les enregistrements recupérés utilisent la table du format MARC-8 pour représenter les caractères spéciaux et diacritiques.  L'équivalence entre les diacritiques et l'alphabet ANSI est contrôlé par l'option Configurer le client Z39.50 dans le menu principal de ABCD.
