Cette procédure vous permet de créer un format de sortie - ou d'utiliser un format précédemment crée - pour envoyer l'information directement à un traitement de texte.

Les enregistrements peuvent être sélectionnés de deux façons

  1. Par une série d'enregistrements (de - à)
  2. En appliquant une équation de recherche à la base de données. Pour cela, cliquez sur l'icône de recherche et rédigez l'équation de recherche

Une fois les enregistrements choisis, sélectionnez un format existant ou créez en un nouveau.

Pour sélectionner un format existant, consultez la liste affichée sous Sélection de formats existants : puis, cliquez sur le format souhaité. Cliquez ensuite sur l'icône pour faire apparaître une fenêtre avec le format choisi. Si vous connaissez le langage de formatage CDS/ISIS, vous pouvez éditer le format afin de le modifier ou de l'ajuster. Dans le cas contraire, ne modifiez pas le format sous peine d'erreurs et vous n'obtiendrez pas de résultats lorsque vous génèrerez un listing.

Cliquez sur Envoyer afin d'appliquer le format au fichier de sortie. Vous pouvez tester ce format soit en sélectionnant les enregistrements par leur MFN ou via une recherche sur la base de données. Une fois la sélection effectuée, cliquez sur Envoyer vers Word (afin d'envoyer le test au traitement de texte) ou sur Prévisualiser pour afficher la sortie du format à l'écran.

Pour créer un nouveau format sélectionnez les champs que vous souhaitez inclure dans la sortie depuis la liste. Pour cela, double-cliquez sur le nom ou sélectionnez à l'aide de la souris et cliquez sur les flèches correspondantes. Vous pouvez utiliser les touches [Ctrl] et [Maj] pour sélectionner plusieurs champs. Les flèches transfèrent simplement les champs sélectionnés d'une fenêtre à l'autre, et la double flèche transfère tous les champs.

Vous pouvez utiliser les boutons haut ou bas pour changer la position des champs sélectionnés.

Une fois votre sélection terminée, cliquez sur Générer le format et le format produit sera affiché dans la fenêtre. Si vous connaissez le langage de formatage CDS/ISIS, vous pouvez éditer le format afin de le modifier ou de l'ajuster. Dans le cas contraire, ne modifiez pas le format sous peine d'erreurs et vous n'obtiendrez pas de résultats lorsque vous génèrerez un listing.

Vous pouvez tester ce format soit en sélectionnant les enregistrements par leur NFM ou via une recherche sur la base de données. Une fois la sélection effectuée, cliquez sur Envoyer vers Word (afin d'envoyer le test au traitement de texte) ou sur Prévisualiser pour afficher la sortie du format à l'écran.

Pour sauvegarder le format créé, donnez un nom (lettres et chiffres uniquement, sans espace) et ajoutez une description qui présentera ce format dans liste des formats enregistrés.
