Formulaires de saisie

Le système propose un formulaire de saisie par défaut basé sur la TDC. D'autres formulaires peuvent être produits ne comprenant que certains champs alors que les autres propriétés du formulaire sont issus directement de la TDC

Création d'un formulaire de saisie (FMT)

La création d'un formulaire de saisie supplémentaire est possible par le menu  Feuille de travail  en haut de la fenêtre ABCD.

Création d'un nouveau  formulaire de saisie:

Sélectionner à partir de la liste des champs ceux qui feront partie du formulaire. Pour les sélectionner :

  • Double-cliquer sur le champ concerné
  • Cliquer sur un champ et étendre la sélection avec les touches  [Maj]  ou  [Ctrl-Maj]  puis cliquer sur la flèche pointant à droite
  • Pour copier tous les champs du formulaire de saisie de données, cliquer sur  la double flèche pointant à droite 

Pour supprimer des champs du formulaire de saisie :

  • double-cliquer sur  le champ concerné
  • Cliquer sur un champ puis étendre la sélection avec les touches [Maj] ou [Ctrl-Maj] puis cliquer sur la flèche pointant à gauche 
  • Pour supprimer tous les champs du formulaire, cliquer sur la double flèche pointant à gauche 

Les champs éliminés du formulaire se retrouvent dans la liste des champs

Utiliser les boutons  ou  pour positionner les champs dans le formulaire de saise.

Utiliser le lien Prévisualisation pour avoir une présentation du formulaire en construction.

Quand c'est terminé, sauvegarder le formulaire après avoir défini dans le champ  nom, le nom du fichier à enregistrer et dans le champ  description mettre une brève description du nouveau format. Le  nom du format   ne peut contenir que des caractères alphabétiques, numériques et le caractère _. Il doit toujours commencer par un caratère alphabétique. Les espaces et autres caractères spéciaux ne sont pas autorisés. La  description doit être raisonnablement courte pour être vue correctement dans le menu  Feuilles de travail en haut de la fenêtre principale de  ABCD.

Modifier un formulaire de saisie existant

Sélectionner à partir de la liste affichée le formulaire à éditer et cliquer sur  Editer. La fenêtre sera remplie avec les informations des champs sélectionnés du formulaire. Suivre les instructions mentionnées plus haut pour l'ajout ou la suppression de champs.

Supprimer un formulaire de saisie existant

Selectionner à partir du menu le formulaire à supprimer et cliquer sur Supprimer.

 

Les formulaires édités sont stockés dans le répertoire def du dossier Base de donnée, avec l'extension .fmt. la liste des formulaires disponibles est conservée dans le fichier  formats.wks dans lequel chaque formulaire prend une ligne suivi du caractère  | qui sépare le nom de sa description.

Exemple d'un fichier de formulaires de saisie:

H||Entrées principales|0|0|||||||||||
100
110
111
240
242
245
246
250
260
300
310

Comme on peut le voir, le formulaire créé comporte les lignes de la TDC du type H (intitulé) ou L (Ligne) et les étiquettes des champs sélectionnés. Cela signifie que les changements opérés dans la TDC  après la création du formulaire de saisie, seront effectifs au moment de la sélection du formulaire correspondant.

Le fichier  formats.wks   est lu pour être affiché dans le menu  Feuilles de travail. Lors de la création ou de la suppression d'un bordereau, le menu Feuilles de travail en haut de la fenêtre principale de ABCD est mis à jour automatiquement automatically.
