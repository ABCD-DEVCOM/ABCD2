Cette option élimine un ordre de paiement émis.

Pour que cette action soit mise en oeuvre, il est nécessaire de retirer les données associées à la recommandation pour l'ordre émis.

L'ordre d'exécution du processus est le suivant :

  1. Accéder aux recommandations relatives au bon de commande
  2. Supprimer les données dans les recommandations du bon de commande
  3. Continuer ce processus pour toutes les recommandations associées au bon de commande 

Etant donné qu'il n'y a pas de lien entre les bons de commande et les recommandations, il permet alors la suppression de la commande
