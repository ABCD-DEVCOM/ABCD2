Capture de termes normalisé à travers d´un tableau

Cette option d'ABCD permet créer des tableaux de valeurs qu'ils seront présentés au moment de l'entrée du champ correspondant. La forme comme s´expose le tableau dépendra du Type d'entrée assigné au champ dans la FDT (checkbox, option, select single, select multiple). Les tableaux de valeurs se trouvent dans le classeur def de la base de données sélectionnée.

Pour créer un nouveau tableau, placez le nom dans l´etiquette Nom, avec l'extension .tab et cliquez sur éditer/créer.
Pour visualiser un tableau existant, cliquez sur l'image ou sur le nom du tableau correspondant.

Pour éditer un tableau existant, cliquez sur le nom même puis cliquez sur éditer/créer.

Pour assigner à un champ un tableau existant, cliquez sur le nom même puis cliquez sur metre à jour la FDT.

Pour éliminer un tableau existant, cliquez sur le nom même puis cliquez sur éliminer. Le tableau sera éliminé du classeur, mais il restera associé au champ respectif dans la FDT. Rappelez-vous alors, éliminer l'entrée dans la FDT.

La liste de tableaux disponibles présente tous les dossiers avec l'extension .tab, ils sont localisé dans le classeur def de la base de données sélectionnée.

Création/Édition de tableaux

Une fois que le nom du dossier a été placé dans le space Nom, cliquez sur créer/éditer. La sélection apparaîtra dans un tableau avec deux colonnes: Code| Terme

Si est rempli seulement la colonne terme, dans la procédure d'entrée ils apparaîtront les valeurs placées dans cette colonne, lequel seront transféré au champ correspondant en utilisant la méthode de la sélection appropriée au type d'entrée du champ défini dans FDT.

Si vous voulez transférer le code et le terme associé, alors la structure du champ qui recevra la valeur du tableau devrait contempler 2 sous-champs. Dans le premier sous-champ sera placée la valeur qui correspond au code et dans le deuxième sous-champ sera placée la valeur qui correspond au terme. Dans la procédure d'entrée ils apparaîtront seulement les valeurs présentes dans la colonne terme, mais le code sera entreposé plus le terme en utilisant les sous-champs correspondantes.

Quand l'édition du tableau est finie, cliquez sur le bouton Metre à jour pour entreposer le tableau créé/modifié et l'assigner au champ qui correspond de la FDT.

Aide: Pour utiliser dans une base de données A, un tableau déjà défini pour une base de données B, précédez le nom du Picklist avec le chemin absolu où le tableau que vous voulez utiliser est localisé.
