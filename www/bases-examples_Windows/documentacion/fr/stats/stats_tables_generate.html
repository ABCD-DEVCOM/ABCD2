Avant d'utiliser cette option vous devez définir les variables de la base de données impliquées dans les statistiques et, éventuellement, les tableaux de fréquence et / ou éventualité où elles s'appliquent.

Pour définir les variables impliquées dans le processus de la statistique, utilisez l'option Modifier définition de base de données du ménu de démarrage du module de gestion

Génération de statistiques

Utiliser une table existante

Présente un menu avec des tableaux pré-définis dans l'optionModifier la définition de base de données. Ensuite, sélectionnez la table que vous souhaitez utiliser

Création d'une table

La liste des champs dans la base de données ont été sélectionnées pour la collecte de statistiques. Sélectionnez le champ à utiliser pour les lignes de la table et, éventuellement, le domaine à utiliser dans les colonnes.

Conseil: Placez rangs comme le domaine qui détient la plus grande diversité de valeurs.

Générer de sortie

Cette section indique le mode de sélection des dossiers pour lesquels ils souhaitent obtenir une matrice de fréquence ou de contingence.

  • Pour rang Mfn
  • Rechercher

Si vous choisissez recherchez vous serez dans un pop-up, le formulaire de recherche avancée pour aider à la construction du terme de recherche appliquée à la   base de données, d'extraire les dossiers dont les champs doivent être quantifiés.

Une fois construit l'expression, cliquez sur  pour envoyer l'expression de recherche à la forme de statistiques.

Vous devez choisir une seule méthode pour extraire les dossiers: ou la recherche ou par Mfn.

Les données sont fournies dans le formulaire, cliquez sur Soumettre pour obtenir les résultats.

 

 

 

Génération de la table

Avec les valeurs fournies dans le formulaire de statistiques la matrice de fréquence ou de contingence demandée est construite.

Avec le déploiement de la table afficher   les options pour envoyer le résultat à l'un des suivants:

  • Une feuille de calcul (.xls)
  • Un document (.doc)
  • Générer un graphique animé
  • Générer un non-graphique animé

Feuille de calcul (Spreadsheet)

Il construit la table et ouvre le programme associé, le client, l' .xls

Document

Il construit la table et ouvre le programme associé, le client, un fichier. doc

Animation graphique

Les résultats sont envoyés à un plug-in générateur de graphiques animés.

Pas de graphiques animés

Les résultats son envoyés à un plug-in générateur, pas de graphiques animés.
