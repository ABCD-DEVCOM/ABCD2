Si l'étape ci-dessus a été achevée, ABCD a déjà enregistrées le FDT et le FST de la base de données en processus de création.

Utilisez ensuite le cadre Télécharger .pft pour trouver le format de sortie pour être affecté à la base de données. Ce processus ne peut aller jusqu'à un format, mais ensuite, par l'intermédiaire du module "Maintien de la structure" vous pourrez soulever d'autres formats.

Rappelez-vous que le format choisi doit être codé de sorte que la production joue un page HTML. Formats qui ne répondent pas à cette exigence vont générer une erreur de syntaxe sous ABCD.
