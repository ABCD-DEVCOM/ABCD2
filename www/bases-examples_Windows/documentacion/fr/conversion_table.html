Table de conversion pour l'export de la base de données vers un fichier TXT avec étiquettes

La table de conversion est utilisée pour l'export et pour l'import des enregistrements, selon les caractéristiques suivantes :

Utilisez le choix Délimité avec tabulations pour importer une feuille Excel sauvegardée au format TXT. Dans ce cas, la colonne Etiquette dans le fichier TXT doit contenir le numéro de la colonne (commençant à 1) où se situe le champ correspondant dans la feuille Excel.

La table de conversion contient les colonnes suivantes  :

Etiquette dans le fichier TXT

Etiquette qui identifie le champ d'entrée ou sortie du fichier TXT. Les étiquettes doivent commencer par $$ et se terminer par :

Etiquette

Etiquette du champ dans la base de données

Sous-champs

Séparateurs de sous-champs. Si le premier sous-champ n'a pas de séparateur, laissez un espace vide

Remplacer par

Si un champ a des sous-champs, indiquez ici les caractères qui seront utilisés dans le texte pour remplacer les délimiteurs de sous-champs. Le premier caractère de cette liste remplacera le premier sous-champ dans la liste. Si le premier délimiteur ne doit pas être remplacé, laissez un espace vide

Séparateur d'occurrences

Indiquez ici le caractère qui séparera les occurrences d'un champ répétitif

Format d'extraction

Ce format s'applique à l'enregistrement afin de produire l'extraction du champ associé. L'application du format n'est exécutée que lors du processus d'extraction. Si vous définissez des délimiteurs de sous-champs en plus du format d'extraction, le dernier est celui alors appliqué durant le processus (pas encore mis en œuvre).
