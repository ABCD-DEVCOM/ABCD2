Sélection d'une base de données

Lorsque vous sélectionnez une base de données, le nombre d'enregistrements est affiché. Il est également précisé si la base de données est verrouillée.

Si la base de données est verrouillé, elle ne peut pas être accessible par tout utilisateur, jusqu'à ce que son administrateur ne l'ait déverrouillée.

Pour verrouiller une base de données utiliser l'option Utilitaires->verrouiller la base dans le menu de démarrage de ABCD.

Seul l'administrateur du système ou l'administrateur de la base de données peut verrouiller ou déverrouiller les bases de données.

Une autre raison pour laquelle une base de données peut être verrouillée est une erreur possible lors de sa manipulation, par exemple, si la procédure d'inversion de fichier n'est pas terminée, ou s'il y a une erreur dans la TSC lors du chargement d'un enregistrement. Cette situation est aussi lors de l'ouverture de la base. Dans ce dernier cas, parce que le verrouillage a été produit par un administrateur, il est nécessaire, d'abord, de déverrouiller la base et ensuite les enregistrements par par l'intermédiaire du menu associé au bouton “Utilitaires” de la barre d'outils, disponible pour le processus de catalogage.

Le verrouillage des enregistrements est produit généralement au cours de l'opération de saisie de données. Ainsi, quand un utilisateur demande à modifier un enregistrement, celui-ci est verrouillé pour éviter des changements indésirables par un autre utilisateur. Lorsque le dossier est mis à jour, il est déverrouillé et mis à disposition pour le reste des utilisateurs.

Si le cycle de mise à jour n'est pas terminé, c'est à dire, si l'enregistrement n'est pas modifié ou si la modification est annulée (boutons Enregistrer ou Annuler de la barre d'outils), l'enregistrement restera verrouillé pendant un certain temps (4 heures) pour le reste des utilisateurs.

Il y a deux moyens pour déverrouiller un enregistrement avant que le délai n'expire :

  1. L'utilisateur qui l'a verrouillé peut le libérer (ou déverrouiller) par une mise à jour ou une annulation de ses changements.

  2. En utilisant l'option “Déverrouiller les enregistrements” diponible dans le menuUtilitaires de la barre d'outils. Uniquement l'administrateur du Système ou l'administrateur de la base de données ont accès à cette option.
