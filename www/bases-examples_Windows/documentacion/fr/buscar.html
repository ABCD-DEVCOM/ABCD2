Recherche avancée 

Le formulaire de recherche avancée peut être défini dans la section Définition des bases de données. Il contient les champs qui ont été indexés dans la TSC et qui peuvent être recherchés via le dictionnaire des termes.

Pour faire une recherche avancée, il faut effectuer les étapes suivantes :

  1. Sélectionner le champ sur laquelle la recherche est effectuée

  2. Cliquer sur Index pour consulter le dictionnaire des termes du champ

  3. Sélectionner les termes souhaités du dictionnaire. Il y a deux façons pour cela :

    • Sélectionner les termes et cliquer sur Recherche pour appliquer directement la recherche aux termes sélectionnés

    • Sélectionner les termes et cliquer sur Envoyer les termes choisis afin de transférer les termes au formulaire de recherche pour construire une expression de recherche qui combine des informations provenant de plusieurs champs

  4. Si vous ne souhaitez pas utiliser le dictionnaire, écrivez directement les mots clés pour retrouver l'information que vous souhaitez en utilisant la boite de dialogue correspondante. Afin de pouvoir utiliser cette option, il est nécessaire d'avoir indexé les termes en utilisant la technique 4 (par mots) dans la TSC. Si vous ne sélectionnez pas de champ dans la liste des champs disponibles, alors la recherche sera appliquée à tous les champs de la base de données ayant été indexés.

  5. Une fois la sélection terminée, cliquer sur afin d'appliquer la recherche à la base de données et de trouver les enregistrements, ou sur pour effacer les termes de recherche.

Les enregistrements trouvés sont présentés dans la fenêtre de la base de données permettant leur édition.
