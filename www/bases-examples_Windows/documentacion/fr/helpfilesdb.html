Fichiers d'aide pour les champs des bases de données

Pour définir les pages d'aide pour les champs des formulaires de saisie, suivez les étapes suivantes :

  1. Sélectionnez dans le menu Table de Sélection des Champs (TSC) le champ pour lequel vous voulez créer une page d'aide 
  2. Après sélection, les propriétés du champ seront affichées telles que définies dans la Table de Définition des Champs (TDC)
  3. Cliquez sur prévisualiser pour afficher la page d'aide, si elle est déjà disponible
  4. Cliquez sur Editer une page d'aide pour créer ou modifier une page existante. L'éditeur HTML s'ouvrira, vous permettant d'écrire le texte de la page d'aide.
  5. Quand l'édition de la page d'aide est prête, cliquez sur Sauvegarder la page pour stocker la page dans l'aide du système.
  6. Les pages d'aide pour les champs sont stockés dans /bases/dbn/aide/tag_xxx.html, où dbn est le nom de la base de données active, et xxx  représente l'étiquette du champ pour lequel l'aide est créée. Ce nom sera transmis automatiquement au système et ne devrait pas être modifié, car il servira de référence à la saisie de données en fournissant une page d'aide.
  7. Quand les pages d'aide sont prêtes, modifiez la TDC de la base pour indiquer, dans la colonne corespondante, pour quels champs les pages d'aide ont été définies.
