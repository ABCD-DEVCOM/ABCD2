Fonctions de maintenance

Elles permettent :

  1. d'initialiser une base de données

  2. de supprimer une base de données

  3. de verrouiller une base de données

  4. de déverrouiller une base de données

Initialiser une base de données

Cette option supprime - de façon définitive et irréversible - tous les enregistrements de la base de données.

Supprime une base de données

Cette option supprime - de façon définitive et irréversible - tous les fichiers de la base de données, formats et fichiers de définition d'une base de données.

Verrouiller une base de données

En utilisant cette option, l'administrateur bloque l'accès à une base de données, que ce soit pour la réorganiser, la génération des index, ou pour n'importe quelle autre raison qui ne permet pas la saisie de données.

Déverrouiller une base de données

En utilisant cette option, l'administrateur autorise l'accès a une base de données qui était verrouillée pour des raisons de maintenance.
