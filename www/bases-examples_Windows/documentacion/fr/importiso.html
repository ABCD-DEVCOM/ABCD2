Charger un fichier ISO

Les fichiers ISO créés dans l'opération d'exportation sont stockés dans le répertoire ./bases/wrk. Pour importer un fichier ISO, cliquez sur l'option Select qui apparaît avec le fichier ISO que vous souhaitez importer.

Si vous voulez remplacer la totalité de la base avec les enregistrements importés, cochez la case nommée Effacer tous les enregistrements avant l'importation. Si vous ne cochez pas cette case, tous les enregistrements importés seront chargés à la suite du dernier enregistrement de la base.

Si vous voulez mettre à jour le fichier inversé pendant l'importation, cochez la case appelée Créer le fichier inversé. Si vous ne sélectionnez pas cette option, vous pourrez l'activer ultérieurement avec une option pour de la création du fichier inversé dans le menu Maintenance de la base de données.

Note importante.: Si la base de données a beaucoup d'enregistrements, il est recommandé d'utiliser l'utilitaire MX pour la création du fichier inversé.

Si le fichier ISO que vous voulez importer n'est pas dans le serveur, vous devez le télécharger avant de procéder à l'importation. Pour ce faire,utilisez la section charger un fichier ISO affiché dans la fenêtre d'importation. Une fois transféré, vous pourrez importer ce fichier.

Note importante.: le fichier ISO à importer doit avoir les mêmes étiquettes et sous-champs définis dans la base de données parce qu'une TSC d'importation pour faire des changements dans la structure des champs n'est pas disponible.

Si le fichier ISO à télécharger est important, vous avez besoin d'utiliser un FTP ou autre moyen de le transférer sur le serveur.

Utilisez l'option Supprimer fourni avec chaque fichier ISO affiché, pour supprimer du serveur les fichiers ISO qui ont déjà été utilisés.
