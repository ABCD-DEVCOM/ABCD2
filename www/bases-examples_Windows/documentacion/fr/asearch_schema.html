Pour cela, une fenêtre divisée en deux cadres est affichée avec les détails suivant :

Cadre de gauche :

Demande les champs qui constitueront le formulaire de recherche avancée. Affiche les colonnes suivantes :

Nom du champ

Tel qu'il apparaît dans le formulaire de recherche avancée

N° dans la TSC(FST)

N° utilisé pour indexer le champ dans la TSC(FST). Si plus d'un champ est inclus dans la même recherche, séparer les différents N° par une virgule. Exemple : 100,600,700

Préfixe

Préfixe assigné au champ afin de présenter les termes de l'index d'un champ ou d'un groupe de champs, en particulier.

Il est important de noter qu'il y a deux façons d'obtenir les termes qui seront inclus dans n'importe quelle zone du formulaire de recherche :

  • Si la sélection est faite via le dictionnaire (lien Index), alors les termes apparaissent dans la zone de recherche avec le préfixe qui leur a été assigné. Si plusieurs termes sont sélectionnés, alors ils seront liés avec l'opérateur logique “or”. Les termes apparaitront entre guillemets (“..”) pour indiquer au processus de recherche de ne pas les séparer mais de les traiter tels qu'ils apparaissent dans le formulaire. Dans ce cas, l'équation sera : "AU_GARCIA MARQUEZ, GABRIEL" or "AU_VARGAS LLOSA, MARIO"

  • Si le terme est directement saisi dans une zone de recherche du formulaire, alors l'équation de recherche sera construite en liant les mots à l'aide de l'opérateur (F) et un opérateur est ajouté à l'expression finale, comme ceci : /(N° dans la FST) utilisant la deuxième colonne mentionnée dans le formulaire de recherche. Pour que cette possibilité fonctionne correctement, les champs correspondant doivent être indexés avec la technique n°4, produisant le nombre d'occurrences avec l'opérateur '%' quand le champ est répétitif. Dans ce cas, l'équation sera générée de la façon suivante : GARCIA(F)MARQUEZ(F)GABRIEL/(100,600,700) or VARGAS(F)LLOSA(F)MARIO/(100,600,700)

Les lignes suivantes sont requises dans la TSC(FST) :

100 0 "AU_"v100^a
600 0 (|AU_|v600^a|%|/)
700 0 (|AU_|v700^a|%|/)
100 4 v100^a
600 4 (v600^a|%|/)
700 4 (v700^a|%|/)

et le formulaire de recherche avancée indiquera :

Nom du champ : Auteur
N° de la TSC(FST) : 100,600,700
Préfixe : AU_

Le formulaire de recherche avancée sera stocké dans le dossier def de la base de données, avec le nom champsrecherche.tab
