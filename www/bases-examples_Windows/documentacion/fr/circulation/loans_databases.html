Base de données des articles disponibles pour le prêt

Les bases de données conçues sous ABCD contenant des articles disponibles pour le prêt peuvnet être sélectionnés par cette procédure.

Choisissez une base de données à partir de la liste affichée et cliquez sur Allez pour définir les paramètres nécessaires et les formats pour relier la base de données au système de prêt.

Dans l'ensemble vous devez définir

  • La clé pour accéder aux titres
  • Les outils pour déterminer le nombre total de copies du titre
  • Comment extraire l'information à partir de chaque copie
  • Comment afficher l'information catalographique du titre
  • Quelle information doit être transmise à la base de données des prêts pour identifier les articles empruntés et pour obtenir les statistiques
  • Comment afficher l'information sur un article emprunté du moment ou il a été sauvegardé dans la base de données des prêts.

Après avoir sélectionné la base de données, un formulaire d'entrée sera affiché vous invitant à entrer les paramètres et les formats requis par la procédure.
