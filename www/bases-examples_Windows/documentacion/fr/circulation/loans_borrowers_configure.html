Configuration de la base de données des usagers

Cette option est utilisée pour configurer le système de base de données avec les usagers habilités à accéder au service de prêt

Paramètres requis

Préfixe pour obtenir le numéro de l'usager Il indique le préfixe à mettre devant le numéro de l'usager pour localiser l'information appropriée dans la base de données des usagers. Il est stocké dans le dossier de la base de données des usagers def sous le nom. loans_uskey.tab
PFT pour extraire le numéro de l'usager Ce format d'extraction est appliqué pour obtenir le numéro de l'usager une fois le registre des usagers a été trouvée. Il est stocké dans le dossier de la base de données des usagers def sous le nom loans_uskey.pft
PFT pour extraire le type d'utilisateur Les politiques de prêt sont appliquées selon le type d'usager et le type d'article à emprunter. Ce format d'extraction est appliqué pour extraire le type d'usager. Il est stocké dans le dossier de la base de données des usagers def sous le nom loans_ustype.tab
PFT pour extraire la validité de l'usager

La validité de l'usager peut être configurée en appliquant ce format.Il doit contenir un message et un code séparé par le caractère.. |
Les codes suivants ont été établis :

0=Usager valide
1=Usager non valide

Il est stocké dans le dossier de la base de données def sous le nom loans_usvig.tab
PFT pour montrer l'information de l'usager Ce format d'extraction format montre l'information de l'usager afin de forunir l'état des comptes et d'autres rapports relatifs à la gestion des prêts. Il est stocké dans le dossier de la base de données des usagers def sous le nom loans_usdisp.tab
