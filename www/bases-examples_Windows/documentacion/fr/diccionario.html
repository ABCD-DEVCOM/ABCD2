Dictionnaire des termes Le dictionnaire des termes du champ sélectionné est affiché. Il est généré à partir de la Table de Sélection des Champs de la base de données. Pour atteindre un autre terme, taper quelques lettres dans la boîte correspondante et cliquer sur Continuer. Pour continuer à naviguer dans la liste à partir du dernier terme affiché, cliquer sur Plus de termes. Choisir un ou plusieurs termes dans la liste. Cliquer sur Rechercher afin de trouver les enregistrements qui correspondent à votre sélection ou sur Envoyer les termes choisis afin de placer les termes sélectionnés dans le formulaire de recherche. Les termes seront entre guillemets indiquant ainsi qu'ils doivent être utilisés tels qu'ils apparaissent. Si vous sélectionnez plus d'un terme, ils seront liés par l'opérateur booléen OU.
