Changements globaux

Pour effectuer des changements globaux, suivre les étapes suivantes:

  1. Choisir la méthode de sélection des enregistrements

  2. Choisir le champ auquel vous voulez appliquer le changement

  3. Choisir le type de changement:

    • Ajouter

    • Modifier

    • Supprimer

    • Diviser

  1. Taper la valeur à changer, par exemple, si vous voulez supprimer le champ 30, quand son contenu est "Médicament".

  2. Si le type de changement sélectionné est Modifier, sélectionner la portée de la modification:

    • Sur tout le champ

    • Sur une partie du champ

  3. Choisir la nouvelle valeur à ajouter ou à modifier

  4. Si le type de changement est Diviser, sélectionner:

    • le(s) caractère(s) à utiliser comme délimiteur(s)

    • le champ où la partie sélectionnée sera placée

    • la partie du champ à déplacer dans le champ sélectionné

Méthode de sélection du champ

Il y a deux manières de sélectionner les enregistrements où le changement global sera effectué:

  1. En précisant l'intervalle de Mfn

  2. En construisant une expression de recherche. Dans ce cas, la recherche avancée est proposée pour construire l'expression de recherche.

Sélectionner la méthode de recherche qui permettra de sélectionner les enregistrements auxquels le changement s'appliquera.

Champ où le changement se fera

Les champs définis dans la TDC apparaissent dans le menu. Sélectionner:

Type du changement

Il y a trois types de changement:

  1. Ajouter le champ

  2. Modifier le contenu d'un champ

  3. Supprimer un champ

Le changement sélectionné sera appliqué au champ sélectionné des enregistrements correspondant à l'intervalle de Mfn spécifié ou en exécutant une recherche.

Portée du changement

Si vous avez sélectionné l'option Modifier le contenu d'un champ, vous aurez besoin de spécifier si vous voulez remplacer le contenu du champ tout entier, ou si le changement sera appliqué à une partie du champ, par exemple, ne remplacer que le contenu d'un sous-champ.

Appliquer un changement au contenu d'un champ

Cette case vous permet de dresser une liste de termes qui pourra conditionner l'exécution du changement. C'est-à-dire, si le contenu du champ coïncide avec au moins un des termes spécifiés, alors, et dans ce cas seulement, le changement s'effectuera. Si vous ne spécifiez aucune valeur, le changement s'appliquera sans aucune condition.

Nouvelle valeur du champ

Taper dans cette case la valeur que vous voulez ajouter ou utiliser comme remplacement pour le champ spécifié.

Diviser un champ

Cette option vous permet de mettre une ponctuation ou d'autres délimiteurs dans un champ afin de :

  • Le transférer vers un autre champ

  • Le supprimer

Vous pouvez sélectionner quelle partie du champ (avant ou après le délimiteur) sera supprimée ou transférée vers un autre champ.

Exécution du changement

Le changement s'exécute pour le groupe d'enregistrements.. Un lien aux enregistrements apparaîtra pour permettre la révision pendant le traitement.

Note importante: Faire une sauvegarde de la base avant de lancer des changements globaux.
