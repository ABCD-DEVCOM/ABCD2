Carga de un archivo Iso

Los archivos ISO que se generan mediante exportación se almacenan en la carpeta ./bases/wrk. Para importar un archivo ISO haga clic sobre la opción Seleccionar que acompaña el ISO que desea importar.

Si desea sustituir la base de datos con el resultado de los registros importados, marque la casilla identificada como Eliminar todos los registros antes de importar. Si no se selecciona esta casilla, los registros importados serán cargados al final de la base de datos.

Si desea actualizar la lista invertida conforme importa los registros, marque la casilla identificada como Generar lista invertida. Si no selecciona esta casilla, deberá luego activar la generación de la lista invertida mediante la opción prevista para tal fin en el menú de Mantenimiento de la base de datos.

Nota importante.: Si la base de datos tiene muchos registros, se recomienda utilizar el utilitario MX para la generación de la lista invertida

Si el archivo ISO que se desea importar no está en el servidor, debe primero subirlo antes de proceder a la importación. Para ello utilice la sección Subir archivo ISO que se muestra en la hoja de importación. Una vez transferido el archivo, puede importarlo.

Nota importante.: el archivo iso a importar debe tener las mismas etiquetas y subcampos de la base de datos ya que no existe una FST de importación para realizar cambios en la estructura de los campos.

Si el archivo iso a subir es de gran tamaño, debe utilizar FTP u otro medio para subirlo al servidor.

Use la opción Eliminar que acompaña a cada archivo ISO presentado, para borrar del servidor aquellos ISO que ya hayan sido utilizados.
