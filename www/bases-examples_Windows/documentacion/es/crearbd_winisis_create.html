Crear una base de datos copiando las estructuras ya existentes en WinIsis

Se puede crear una base de datos subiendo la FDT, FST y PFT existentes en una definición local bajo WinIsis.

Para ello ABCD permite subir la FDT desde la máquina local y la convierte a la estructura de FDT mantenida bajo este sistema.

Igualmente permite subir la FST y un formato de salida (PFT).

Tenga en cuenta que el formato de salida seleccionado debe estar escrito utilizando pre y post literales que reproduzcan correctamente una página HTML. Formatos de despliegue que contengan los elementos del lenguaje de formateo definidos para la visualización en WinIsis y no en un browser, producirán error de sintaxis al ser visualizados bajo ABCD.

Una vez cumplidos los tres pasos ABCD creará la base de datos y la adicionará a la lista de bases de datos existentes.

Para que la base de datos pueda ser operada por otros usuarios que no tengan permiso de "Administrador del Sistema", es necesario entrar a la opción de Administración de usuarios, crear o seleccionar los usuarios que pueden acceder a ella y darle la permisología adecuada.
