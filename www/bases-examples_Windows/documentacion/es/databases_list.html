Este procedimiento le permite cambiar el órden en el cual aparece la lista de bases de datos habilitadas en el sistema.

Seleccione la base de datos  y haga clic sobre o   hasta colocarla en el lugar correspondiente.

Una vez finalizado el proceso, haga clic sobre Actualizar para almacenar la nueva lista con los cambios aplicados.

Debe volver a entrar al sistema para que los cambios se muestren en la lista de bases de datos colocada en el marco superior.

La lista de bases de datos se mantiene en el archivo bases.dat localizado en la carpeta bases.
