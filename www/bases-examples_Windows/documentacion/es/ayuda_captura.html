Capturar registros de otras bases de datos

Seleccione la base de datos desde la cual va a capturar el registro. Use la lista de selección que se encuentra en el recuadro Capturando desde la base de datos:. Esta selección permanecerá activada hasta que la cambie por otra base de datos

Una vez realizada la selección de la base de datos, en el recuadro derecho se le mostrará la lista alfabética de la entrada principal de los registros de la base de datos correspondiente, la cual debe especificarse en la Tabla de Definición de Campos . Haga clic sobre una entrada para presentar su detalle.

Para moverse en la lista de registros, use el Índice alfabético o haga clic sobre Más términos para continuar con la lectura a partir del último término presentado en la ventana. Si desea ir a un título específico, coloque las primeras letras o palabras del mismo en la ventana Ir a y presione [Enter] o haga clic sobre continuar.

Puede utilizar el borde interior del marco para agrandar o disminuir el ancho de las ventanas

Si quiere copiar a la base de datos activa el registro presentado, haga clic sobre para activar el modo de ingreso. Puede editar los campos antes de copiarlo, y los cambios no se verán reflejados en el registro original.

No cierre esta ventana si va a capturar más de un registro.

La última base desde la cual ejecute una captura de registros, permanecera activa durante los siguientes procesos de captura.

Tabla de conversión

Puede seleccionar una tabla de conversión para hacer equivalencias de campos entre bases de datos con diferente estructuras. Las tablas de conversión se crean o editan bajo la opción Configurar cliente Z39.50 del menú principal de ABCD
