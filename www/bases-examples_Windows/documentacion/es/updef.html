Modificar las definiciones de una base de datos

Una vez que una base de datos ha sido creada, se puede entrar a este menú para solicitar la modificación de cualquiera de los componentes que define una base de datos ABCD.

Los componentes existentes son:

  • Tabla de definición de campos. Define la estructura de los registros de la base de datos 
  • Tabla de extración de campos. Define la estrategia a aplicar para construir los índices para acceder los registros de la base de datos 
  • Formatos de entrada. Formatos de entrada adicionales que pueden utilizarse para el ingreso de información 
  • Formatos de salida. Formatos de despliegue que pueden utilizarse para visualizar la información
  • Tipos de registro. Define los formatos de entrada a utilizar para cada uno de los tipos de registro definidos en la base de datos (si los hubiere)
  • Formatos de validación de registros. Permite hacer una revisión de la consistencia de un registro previo a la actualización del mismo
  • Formulario de búsqueda avanzada. Define los campos que pueden utilizarse para constituir la búsqueda avanzada 
  • Lista de bases de datos disponibles. Lista de bases de datos que se presentan en los diferentes módulos de ABCD 
  • Archivo dbn.par. Camino de acceso hacia los recursos de una base de datos
  • Archivos de ayuda para los campos de la base de datos. Ayudas en línea que pueden desplegarse al momento de ingreso de la información
  • Configuración bajo el IAH. Configuración de la base de datos en el sitio de consulta publica
  • Lista de variables para la obtención de estadísticas. Lista de los campos que pueden utilizarse para la obtención de estadísticas, indicando el formato de extracción de cada uno
  • Cuadros estadísticos. Combinación de campos en filas y columnas para conformar cuadros pre-definidos
