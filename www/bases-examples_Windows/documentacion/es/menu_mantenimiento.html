Funciones de Mantenimiento.

Permiten:

  1. Inicializar una base de datos
  2. Eliminar una base de datos
  3. Bloquear una base de datos
  4. Desbloquear una base de datos

Inicializar una base de datos

Esta opción elimina - en forma permanente e irrecuperable - todos los registros de una base de datos.

Eliminar una base de datos

Esta opción elimina - en forma permanente e irrecuperable - todos los archivos de datos, formatos y archivos de definición de una base de datos

Bloquear una base de datos

Mediante esta opción el administrador puede bloquear el acceso a una base de datos bien sea por que se esté reorganizando, sus índices se estén reconstruyendo o cualquier otra situación que no permite el ingreso de datos

Desbloquear una base de datos

Mediante esta opción el administrador permite el acceso a una base de datos que estaba bloqueada por razones de mantenimiento.

Asignar número de control

Mediante esta opción ABCD puede asignar en forma automática un número de control que se incremeta automáticamente desde un valor inicial.  Necesita que en la FDT de la base de datos esté definido un campo con el tipo autoincrement
