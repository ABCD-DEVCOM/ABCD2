Si el paso anterior fue completado, ya ABCD posee registradas la FDT y la FST de la base de datos en proceso de creación.

Use entonces el recuadro Subir .pft para localizar el formato de salida a ser asignado a la base de datos. En este proceso solo se permite subir un formato, pero luego, a través del módulo de "Mantenimiento de las estructuras" podrá subir formatos adicionales.

Recuerde que el formato seleccionado tiene que estar codificado de forma tal que la salida reproduzca una página HTML. Formatos que no cumplan con esta condición generarán un error de sintáxis bajo ABCD.
