Restaurar el número de inventario o el número de control de una base de datos

Número de inventario

El número de inventario es una numeracion que puede llevar en forma automática ABCD sobre la base de datos de existencias (copies). Este número está definido en la FDT de la base de datos con el tipo de campo autoincrement y se incrementa con cada registro creado, a partir del último valor almacenado.

El último valor almacenado se obtiene de un archivo denominado control_number.cn localizado en la carpeta data de la base de datos en cuestión.

Esta opción modifica el valor del último número asignado a efectos de restablecer la numeración de los registros.

Número de control

El número de control es  un campo obligatorio en las bases de datos bibliográficas cuyas existencias van a ser manejada a través de la base de datos copies, por cuanto constituye el campo de relación entre el objeto y sus copias.

Al igual que en el caso anterior, ABCD puede encargarse de generar en forma automática este número de control a través de una numeración secuencial. Al igual que en el caso anterior, el último número asignado se mantiene en el archivo control_number.cn localizado en la carpeta data de la base de datos en cuestión


Esta opción modifica el valor del último número asignado a efectos de restablecer el valor del número de control de los registros.
