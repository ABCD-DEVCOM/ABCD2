Exportación de la base de datos hacia un archivo TXT con rótulos

Para exportar una base de datos en la forma de un archivo TXT con rótulos, se utiliza una de las tablas de conversión ya definidas en la opción de Importación. (Ver Tablas de conversión)

Para ejecutar la exportación, siga el siguiente procedimiento:

  1. Seleccione la tabla de conversión a utilizar en el proceso. Para ello haga clic sobre la opción Seleccionar que acompaña a cada tabla de conversión presentada en la ventana
  2. Seleccione los registros que desea exportar a través de las opciones previstas:
    • Rango de Mfn
    • Búsqueda
    • Una letra del glosario
  3. Seleccione la salida donde quiere obtener el archivo TXT con los registros exportados, según las siguientes opciones:
  4. Procesador de palabras
  5. Vista previa

Una vez seleccionados los registros a exportar, inserte en el recuadro el nombre del archivo ISO que a generar y haga clic sobre el icono .

Al terminar la generación del archivo ISO se presenta una página con el resultado y un vínculo hacia el archivo ISO generado. Si desea bajar el archivo ISO, colóquese sobre el vínculo y haciendo clic conel botón derecho tendrá acceso a la opción que le permitirá bajar el ISO a su computadora.

Posteriormente, en el proceso de importación, se mostrará el ISO generado como parte de la lista de los archivos residentes en la carpeta ./bases/wrk.

Nota importante.: Si la base de datos tiene muchos registros, se recomienda utilizar el utilitario MX para la generación del archivo ISO
