Configuración de las opciones para aplicar el protocolo Z3950 en los procesos de captura de información

Se presentan tres opciones:

Servers
Define la lista de servidores que van a presentarse en el menú de selección de la hoja de búsqueda de información. Deben suministrarse los siguientes campos

  • Nombre del servidor, tal como se presentará en el menú de selección de servidores Z39.50
  • URL a utilizar para su localización
  • Puerto que está habilitado para el acceso al servidor a través del protocolo Z39.50
  • Nombre de la base de datos que se va a accesar en el servidor remoto
La base de datos de servidores se almacena en la carpeta bases bajo el nombre servers

Formato de conversión

La información de los servidores Z3950 siempre se presenta en formato MARC. Esta opción permite definir un formato de conversión para hacer equivalencias entre los campos MARC de la base de datos Z3950 y los campos correspondientes a la base de datos local.

La tabla de conversión considera dos columnas:
  • La primera columna contiene la etiqueta del campo que va a recibir la información en la base de datos local
  • La segunda columna contiene el campo a aplicar al registro MARC para transferir la información a la base de datos local
La conversión se realiza al momento de transferir un registro a la base de datos local

Los formatos de conversión se almacenan en la carpeta /bases/cnv/. La lista de los formatos existentes se denomina z3950.cnv


Tabla de conversión de caracteres MARC-8 a ANSI

Los servidores Z3950 representan los caracteres especiales utilizando la codificación MARC-8, la cual contempla dos posiciones para cada diacrítico. Esta tabla transforma la codificación MARC-8 (dos caracteres para cada diacrítico) a ANSI (un carater para cada diacrítico).

Si luego de hacer una búsqueda sobre un servidor Z3950 aparecen combinaciones de caracteres que no corresponden al lenguaje  correspondiente, debe incluir dicha combinación en la tabla de conversión a fin de que el caracter correspondiente sea representado correctamente. 

La tabla de conversión de diacríticos se almacena en la carpeta bases bajo el nombre marc8-to-ansi.tab
