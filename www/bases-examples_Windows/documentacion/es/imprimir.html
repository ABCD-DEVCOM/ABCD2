Este procedimiento le permite generar un formato de salida - o utilizar uno previamente generado - para enviar información directamente a un procesador de palabras.

Los registros pueden ser seleccionados mediante dos vías

  1. A través de un rango de registros (desde - hasta)
  2. Aplicando una expresión de búsqueda sobre la base de datos. Para ello haga clic sobre el ícono de búsqueda y construya la expresión correspondiente

Una vez seleccionados los registros, debe proceder a seleccionar un formato existente o a construir un nuevo formato.

Para seleccionar un formato ya existente, revise la lista que se muestra bajo Seleccione un formato ya definido: y haga clic sobre el formato que desea utilizar. A continuación haga clic sobre el ícono lo cual le presentará una ventana con el formato seleccionado. Si conoce las reglas para elaboración de formatos del Cds/isis, puede editar el formato para realizar cambios o ajustes; de otra forma, no modifique la salida ya que si genera errores no obtendrá ningún resultado al ordenar la emisión del listado.

Haga clic sobre enviar para mandar el formato a la hoja de generación de salida. Puede probar el formato seleccionando los registros bien sea por rango de Mfn o realizando una búsqueda sobre la base de datos. Una vez realizada la selección haga clic sobre Enviar a Word (para enviar la prueba obtenida a un procesador de palabras)  o sobre Vista previa para ver el resultado del formato a través de su despliegue por pantalla.

Para generar un nuevo formato seleccione de la lista de campos aquellos que desea incluir en la salida. Para seleccionar los campos de doble click sobre el nombre o resáltelos con ayuda del "mouse" y de clic sobre las flechas correspondientes. Puede utilizar las teclas [Ctrl] y [Shift] para seleccionar los campos. Las flechas simples transfieren los campos seleccionados de una ventana a otra y las flechas dobles transfieren todos los campos.

Puede utilizar los botones subir o bajar para cambiar la posición de los campos seleccionados.

Al terminar la seleccion de clic sobre Generar formato y aparecerá en la ventana el formato generado. Si conoce el lenguaje de formateo de Cds/isis, puede editar el formato para realizar cambios o ajustes; de otra forma, no modifique la salida ya que si genera errores no obtendrá ningún resultado al ordenar la emisión del listado.

Puede probar el formato seleccionando los registros bien sea por rango de Mfn o realizando una búsqueda sobre la base de datos. Una vez realizada la selección haga clic sobre Enviar a Word (para enviar la prueba obtenida a un procesador de palabras)  o sobre Vista previa para ver el resultado del formato a través de su despliegue por pantalla.

Para guardar el formato elaborado suministre un nombre (utilizando letras y números únicamente y sin espacios en blanco) y provea la descripción con la cual se presentará ese formato en la lista de formatos ya definidos.
