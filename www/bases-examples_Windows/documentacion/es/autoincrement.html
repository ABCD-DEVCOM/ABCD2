Generación automática del número de control de los registros de una base de datos.

Esta opción permite asignar en forma automática un número secuencial consecutivo, a los registros de una base de datos.

En la FDT se define el campo de control, como tipo de campo "Autoincrement". En la definición del campo debe incluirse el prefijo con el cual se indiza en la lista invertida y los formatos de presentación y extracción correspondientes.

La numeración se lleva en el archivo control_number.cn el cual se almacena en la carpeta data de la base de datos correspondiente. En ese archivo se almacena el último número asignado en el proceso de ingreso de registros.

En el formulario de ingreso del registro no puede insertarse información directamente en el campo del número de control y la asignación del número puede hacerse por uno de los siguientes métodos:

  1. Si se deja el campo en blanco, al momento de almacenar el registro se determina el próximo número a asignar y el mismo se graba en el campo correspondiente
  2. Si se quiere asignar directamente un número de control se puede seguir el siguiente procedimiento:
    • Haga clic sobre el vínculo Asignar que se muestra contiguo a la casilla de ingreso del campo con el número de control.
    • Se abre una nueva ventana para que se introduzca el número de control a asignar
    • Haga clic sobre Enviar para validar si el número de control no existe.
    • Si la validación es exitosa (el número de control no existe), se pasa el valor al formulario de ingreso
    • Si la validación no es exitosa (el número de control existe) se presenta el registro que lo tiene asignado. Haga clic sobre Regresar para asignar otro número o sobre Cerrar para cerrar la ventana
  3. Para borrar la casilla del número de control, haga clic sobre Asignar, deje el número de control en blanco y luego haga clic sobre Enviar
