Archivos de ayuda para los campos de la base de datos

Para definir las ayudas de campo para el ingreso de información en una base de datos siga los siguientes pasos:

  1. Seleccione del menu Tabla de definición de campos (FDT) el campo al cual le va a definir la ayuda 
  2. Realizada la selección se le presentará en el recuadro las características del campo tal como está definido en la tabla de definición de campos (FDT)
  3. Haga clic sobre Vista previa para presentar la ayuda, si esta ya estuviera presente
  4. Haga clic sobre Editar archivo de ayuda para crear la ayuda o modificar la ya existente. Se presentará el editor de HTML para que Ud. pueda escribir el texto de la ayuda
  5. Al finalizar con la edición de la ayuda, haga clic sobre Guardar ayuda para almacenar el archivo
  6. Los archivos de ayuda de campos se almacenan en la carpeta /bases/dbn/ayudas/tag_xxx.html, donde dbn es el nombre de la base de datos activa y xxx representa al tag del campo al cual le corresponde la ayuda. Este nombre lo suministra el sistema en forma automática y no debe ser modificado ya que servirá de referencia en el procedimiento de ingreso para la presentación de la ayuda.
  7. Una vez completados los archivos de ayuda, debe editar la FDT de la base de datos para marcar, en la columna correspondiente, aquellos campos para los cuales se han definido archivos de ayuda.
