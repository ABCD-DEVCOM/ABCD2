Formato Bibliográfico MARC 21 LITE: Campos de Número de Classificação ou de Chamada (05X-08X)

Os campos 050-08X contém os números de classificação ou de chamada relacionados ao ítem descrito no registro bibliográfico. A identificação do conjunto de caracteres que se utiliza para codificar os dados bibliográficos também é provida em um campo deste grupo.
  • 050 – NÚMERO DE CLASSIFICAÇÃO DA BIBLIOTECA DO CONGRESSO - USA (LC) (R)
  • 060 – NÚMERO DE CLASSIFICAÇÃO DA BIBLIOTECA NACIONAL DE MEDICINA - USA (NLM) (R)
  • 066 – CONJUNTO DE CARACTERES PRESENTES NA OBRA (NR)
  • 080 – NÚMERO DE CLASSIFICAÇÃO DECIMAL UNIVERSAL - CDU (R)
  • 082 – NÚMERO DE CLASSIFICAÇÃO DECIMAL DE DEWEY  - CDD (R)
  • 084 – OUTRO NÚMERO DE CLASSIFICAÇÃO (R)
  • 086 – NÚMERO DE CLASSIFICAÇÃO PARA DOCUMENTOS GOVERNAMENTAIS (R)

[Maiores informações em: Formato MARC 21 Conciso]

072 Código de Categorias de Assunto (R)

Um código para as categorias associadas com o ítem descrito.

 

Indicadores

  • Primeiro – não definido
    • # - não definido
  • Segundo – Código fonte
    • 7 – Código fonte especificado no Subcampo ^2

Subcampos

  • ^a - Código de categoria temática (NR):
    • Código amplo ao tema (assunto) associado ao ítem.
  • ^x - Código de subdivisão de categoria temática (R)
    O nível de especificidade dentro do tema geral do Subcampo ^a.
  • ^2 - Fonte do código (NR)
    Um código MARC que identifica o tesauro utilizado para atribuir o código de categoria de matérias usada quando o segunda posição do indicador que contem valor 7. A fonte para os códigos é Lista de Código MARC para Relatores, Fontes, Convenções de Descrição .

Exemplos

072 #7^aE5^x.510^2mesh
072 #7^a17^x02.1^2cosatisc
