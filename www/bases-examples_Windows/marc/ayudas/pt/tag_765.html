Formato Bibliográfico MARC 21 LITE: Campos de Entrada de Ligações (76X-78X)

Os campos 760-787 contém informações que identificam outros itens bibliográficos. Cada entrada destes campos especifica uma relação diferente entre o item objeto descrito pelo registro e o item relacionado com ele. Estas relações são de três tipos:
  1. itens relacionados que ajudam ao usuário em uma busca mas que não são fisicamente requeridos para obter o item desejado (p.ex., entradas previas para seriados, tradução do item relacionado); itens relacionados que tem que ser obtidos fisicamente para poder utilizar o registro relacionado (p.ex. item hospedeiro que contém parte componente: um fascículo de uma revista que contém um artigo específico);
  2. itens relacionados que são unidades constituintes de uma totalidade maior (p.ex. fotografias individuais contidas em uma coleção de material visual).

Os campos de ligações de entradas têm o objetivo de gerar notas nas saídas nas quais aparecem, prover vinculações eletrônicas entre o registro bibliográfico do item objeto e o registro bibliográfico do item relacionado, no caso de o registro relacionado estar em um registro separado; e/ou facilitar a indexação.

765 Entrada de idioma original (R)

Informação concernente de uma publicação no idioma original quando o item objeto é uma tradução (relação horizontal). Quando uma nota é gerada a partir deste campo, a frase Tradução de: pode ser gerada automaticamente baseada na etiqueta do campo para exibição.

Indicadores

  • Primeiro – Controle de nota
    • 0 – Nota a exibir
  • Segundo – Controle de exibição de uma constante
    • # - Tradução de
