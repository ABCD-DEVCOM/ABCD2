CAMPOS DE CONTROLE 001- 005

Os campos 001-005 contém os números e outras informações codificadas de controle e são utilizadas para processar os registros bibliográficos. Cada campo de controle está identificado por uma etiqueta (tag) no diretório. Contém um só elemento ou uma série de elementos de códigos dentro de um campo de tamanho fixo, identificados pela posição. Os campos variáveis de controle não contém indicadores ou subcampos.

  • 001 - NÚMERO DE CONTROLE (NR)
  • 005 - DATA E HORA DA ÚLTIMA MODIFICAÇÃO (NR)

[Maiores informações em: Formato MARC 21 Conciso]

003 Identificador do Número de Controle (NR)

Código MARC da organização, cujo número de controle está contido no campo 001 (Número de controle). A fonte deste código é o "Symbols of American Libraries" mantido pela Library of Congress. É gerado automaticamente pelo sistema. Ver a  Lista Código MARC para Organizações.

Exemplos
003 DLC
003 OCoLC
003 DNLM
