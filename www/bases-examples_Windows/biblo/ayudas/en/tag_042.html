Etiqueta

 42

Nombre

Información descriptiva

Definición del dato

Información   gráfica   que   acompaña   al   texto   de   un   documento (ilustraciones, gráficos, mapas, etc.).

Propósito del campo

Este campo se utiliza para consignar la información descriptiva de un documento tratado a nivel analítico, monográfico, colección o publicación seriada.

Descripción de los elementos

Este elemento puede consistir en uno o más tipos de información gráfica tales como tablas, cuadros, gráficos, fotografías, mapas, etc.

Esta  información  se  registra  en  forma  abreviada,  utilizando  las siguientes abreviaturas:

ilus.: ilustraciones (incluye fotografías, dibujos, etc.)
diagrs.: diagramas
grafs.: gráficos
maps.: mapas
tbls.: tablas y cuadros

Al registrar más de un tipo de información gráfica, éstas deben registrarse separadas entre sí por coma.

Ejemplo

tbls.

diagrs., maps. ilus.
