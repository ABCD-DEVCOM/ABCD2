Preenchendo
o formulário 		Entre com uma ou mais palavras nos campos do formulário. Se preferir, selecione a pesquisa através de um índice específico, onde as palavras devem ser procuradas.

Se desejar ver ou selecionar a partir de um índice específico, selecione o campo e clique no botão "índice".

Para digitar mais de uma palavra em uma mesma linha é necessário usar um operador booleano para relacionar uma com a outra (exemplo 1). Utilize as linhas 1, 2 e 3 para pesquisar mais de uma palavra e/ou em campos diferentes (exemplo 2)

Clique no botão "Pesquisa" para executar a busca.

Operadores booleanos Para combinar a pesquisa em mais de uma palavra no mesmo campo do formulário, é necessário usar os operadores booleanos entre elas. Abaixo se encontra a tabela do operadores booleanos e exemplos de seu uso:
   
  operador

exemplo

ação
and paciente and cuidados recupera os documentos que contenham as duas palavras : paciente e cuidados.
or paciente or cuidados recupera os documentos que contenham a palavra paciente ou a palavra cuidados ou documentos que contenham as duas palavras: paciente e/ou cuidados ou ambas.
and not paciente and not cuidados recupera os documentos que contenham apenas a palavra paciente e não contenham a palavra paciente: paciente e não cuidados
