Notas :
  • Esta opción busca palabras del título o subtítulo de las revistas.
  • No es necesario usar operadores booleanos (AND, OR o AND NOT) entre las palabras. Seleccione la opción todas las palabras (AND) para relacionar las palabras (restringe la búsqueda), o la opción alguna palabra (OR) para sumar las palabras (amplia la búsqueda).
  • Para buscar por otros campos o para especificar el campo de busqueda, use el formulario de Búsqueda avanzada.
