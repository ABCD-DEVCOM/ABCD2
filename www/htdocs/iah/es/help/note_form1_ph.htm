Notas :
  • Esta opción busca combinando palabras de los nombres o apellidos de propietarios, depositantes, localidades, departamentos y padrones
  • Use el símbolo de truncar $ para buscar palabras con la misma raíz.
    Ejemplo: Pere$ recupera Pereira, Pereyra, Perez, etc. No es necesario usar operadores booleanos (AND, OR o AND NOT) entre las palabras. Seleccione la opción todas las palabras (AND) para relacionar las palabras (restringe la búsqueda), o la opción alguna palabra (OR) para sumar las palabras (amplia la búsqueda).
  • Para buscar por otros campos o número de padrón, use el formulario de Búsqueda avanzada.
