Notas :
  • Esta opción busca palabras del título, palabras del resumen, y descriptores de asunto.
  • Use el símbolo de truncar $ para buscar palabras con la misma raíz. Ejemplo: educ$ recupera educación, education, educação, etc. No és necesario usar operadores booleanos (AND, OR o AND NOT) entre las palabras. Seleccione la opción Todas las palabras (AND) para relacionar las palabras (restringe la búsqueda), o la opción Cualquier palabra (OR) para sumar las palabras (amplia la búsqueda).
  • Para buscar por otros campos o para especificar el campo de busqueda, use el formulario de Búsqueda avanzada.
