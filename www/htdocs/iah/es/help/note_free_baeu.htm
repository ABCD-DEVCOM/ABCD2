Notas :
  • Esta opción busca palabras individuales en los títulos de libros y revistas, conferencias, nombres y apellidos de autores, nombres de instituciones, palabras en los resúmenes, número de expediente,números y artículos de normativa, fecha de edición y temas.
  • Use el símbolo de truncar $ para buscar palabras con la misma raíz.
    Ejemplo: Pere$ recupera Pereira, Pereyra, Pérez, etc. No es necesario usar operadores booleanos (AND, OR o AND NOT) entre las palabras. Seleccione la opción todas las palabras (AND) para relacionar las palabras (restringe la búsqueda), o la opción cualquier palabra (OR) para sumar las palabras (amplía la búsqueda).
  • Para buscar por otros campos de datos o para especificar el campo de busqueda, use el Formulario Básico
